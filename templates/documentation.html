Introducción

Una pequeña aplicación web que busca mediante el uso de IA mejorar la capacidades conversacionales de los estudiantes, haciendo uso de algoritmos de IA como speech-to-text se puede crear un modelo gramatical cerrado que evalúe específicamente las palabras o frases de cada ejercicio aumentando asi la velocidad de procesamiento y evaluación del mismo.

El ejercicio va orientado a llevar al estudiante a no traducir en su cerebro las palabras que ve en pantalla esto último haciendo uso del programa de Neurolinguistica y Neuroplasticidad desarrollado por Salvatore Marrone.

El objetivo de la prueba es permitír que la evaluación de cada ejercicio sea realizada de forma autónoma por un altgoritmo de IA y asi poder masificar esta forma de enseñanza a travez de la inmersión, neurolinguistica y neuroplasticidad.

Compatibilidad:

En esta versión solo se tendrá compatibilidad en navegadores en base a Chrome (Chromium) en versiones actuales.

Sobre este aplicativo

En esta versión se utiliza el reconocimiento de voz en tiempo real para la evaluación del ejercicio

Tecnología

Este aplicativo está construido con las siguientes tecnologías: Python, Flask, Javascript, HTML and CSS.

En el caso de IA se utiliza Speech Recognition de Web Speech API (JS)
